# Projeto-Final

**EcoTrack:**

Aplicativo para acompanhamento de hábitos sustentáveis, focado em consumo consciente de água, energia e resíduos.

**Funcionalidades:**
- Registro de atividades e pontuação diária.
- Desafios do dia (ex.: reduzir tempo de banho).
- Gráficos de progresso para acompanhar evolução.
- Sugestões baseadas no clima usando API (Retrofit).
- Dados locais armazenados com Room.

**Algumas Tecnologias Usadas:**
- Kotlin • Room • Retrofit
